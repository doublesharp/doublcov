# Node Test Runner with c8

Builder names: `v8` (aliases: `c8`, `node`, `node-test`). Runs `npx c8 --reporter=lcov node --test` and feeds the produced LCOV into Doublcov.

Use this for projects that rely on Node's built-in `node:test` test runner with V8 coverage captured by [`c8`](https://github.com/bcoe/c8).

## Prerequisites

- Node 22+ to run the Doublcov npm CLI
- [`c8`](https://github.com/bcoe/c8)

```bash
npm install --save-dev c8
```

## Install

```bash
npm install --save-dev @0xdoublesharp/doublcov
```

Or:

```bash
pnpm add -D @0xdoublesharp/doublcov
```

## Quick start

```bash
doublcov v8
```

Default output: `coverage/report`, unless c8 or Doublcov config resolves a different LCOV/report path. Local runs open the report by default.

## Passing arguments to node --test

The built-in builder wraps `node --test` with c8. Arguments after `--` are passed to `node --test`:

```bash
doublcov v8 -- test/**/*.test.js --test-reporter spec
```

Configure c8-specific options such as `include` and `exclude` in c8 config, or use the manual LCOV path when those flags need to be supplied on the command line.

## package.json scripts

```json
{
  "scripts": {
    "coverage": "doublcov v8",
    "coverage:ci": "doublcov v8 --no-open"
  }
}
```

Doublcov reads c8 coverage defaults from `package.json` `c8.report-dir` / `c8.reportDir`, `.c8rc`, and `.c8rc.json`.

## Manual LCOV path

```bash
npx c8 --reporter=lcov --report-dir coverage --include 'src/**/*.js' --exclude 'src/**/*.test.js' node --test
doublcov build \
  --lcov coverage/lcov.info \
  --sources src \
  --out coverage/report
```

## CI snippet

```yaml
- uses: actions/setup-node@v4
  with:
    node-version: 22
- run: npm ci
- run: npx doublcov v8
- uses: actions/upload-artifact@v4
  with:
    name: coverage-report
    path: coverage/report
```

## Troubleshooting

- **TypeScript sources.** `c8` instruments transpiled output. Either compile first and run tests against the compiled output, or use a loader (`tsx`, `ts-node`) and configure `c8` `include`/`exclude` to point at the original sources.
- **No `lcov.info` produced.** Verify `node --test` exits cleanly. `c8` only writes the report when the wrapped command succeeds (or with `--clean=false`).
- **Custom report directory.** If you change `--report-dir`, put it in c8 config or pass `--lcov` at the matching path.
